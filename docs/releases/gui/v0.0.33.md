Release-Datum: 25.02.2026

- Installer-Anzeige und Textdarstellung mit Umlauten stabilisiert.

- WPF-Startfehler beim Theme-Laden nach Assembly-Umbenennung behoben.
- Release-Workflow vollständig auf dateibasierte Versionshinweise (`docs/releases/vX.Y.Z.md`) umgestellt.

## Release-Prozess

- Altes automatisches Commit-Prefix-Release-Notes-System entfernt.
- Neues System aktiv:
  - Pro Release eine gepflegte Datei in `docs/releases/vX.Y.Z.md`
  - Workflow übernimmt diese Datei automatisch als öffentliche GitHub-Release-Beschreibung
  - Fallback auf Commit-Historie nur, wenn die Datei fehlt
