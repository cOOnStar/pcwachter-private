# Release Notes v0.0.45

Release-Datum: 25.02.2026
Tag: `v0.0.45`

## Kurzüberblick

- Das Setup zeigt bei bestehender Installation jetzt die aktuell installierte und die neue Zielversion.
- Bei identischer Versionsnummer erscheint ein eigener Hinweis, dass bereits die neueste Version installiert ist.
- Anwender können in diesem Fall gezielt entscheiden, ob eine Neuinstallation trotzdem ausgeführt werden soll.

## Setup-Dialog

- Versionsanzeige im Update-Dialog erweitert:
  - „Aktuell installierte Version: …“
  - „Neue Version zur Installation: …“
- Neue Dialogvariante für gleichen Versionsstand:
  - Hinweis „Es ist bereits die neueste Version installiert“
  - Rückfrage zur erneuten Installation
- Fallback zur Versionsauslese aus der installierten `PCWächter.exe`, falls Registry-Wert fehlt.
