# Release Notes v0.0.44

Release-Datum: 25.02.2026
Tag: `v0.0.44`

## Kurzüberblick

- Das Setup erstellt die geplante Aufgabe für automatische Updates wieder korrekt.
- Die Desktop-GUI wird jetzt zuverlässig als Single-Instance ausgeführt.
- Die Hotfixes wurden lokal mit Build- und Smoke-Tests validiert.

## Installer

- Erstellung der geplanten Aufgabe auf einen stabilen Launcher-Aufruf umgestellt.
- Fehler durch fehlerhafte Argument-Quotierung bei `schtasks /Create /TR` behoben.

## Desktop-GUI

- Single-Instance-Schutz erweitert (Mutex + exklusiver Lockfile).
- Mehrfachklicks auf das Desktop-Icon starten keine zusätzliche GUI-Instanz mehr.
