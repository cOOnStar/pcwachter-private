# Release Notes v0.0.50

Release-Datum: 26.02.2026
Tag: `v0.0.50`

## Kurzüberblick

- Update-Erkennung gegen GitHub `latest` wurde robuster gemacht.
- Falsche "bereits aktuell"-Meldungen bei installierter v0.0.48 wurden behoben.
- Versionsauflösung aus Registry wurde für mehrere Schlüssel und Hives vereinheitlicht.

## Update-Prozess

- Die installierte Version wird jetzt aus mehreren Quellen gelesen:
  - `HKLM\Software\Microsoft\Windows\CurrentVersion\Uninstall\PCWaechter` (`DisplayVersion`)
  - `HKCU\Software\Microsoft\Windows\CurrentVersion\Uninstall\PCWaechter` (`DisplayVersion`)
  - `HKLM/HKCU\Software\PCWaechter` (`DisplayVersion` und `Version`)
- Versionswerte mit `v`-Präfix werden korrekt normalisiert.
- Wenn kein Registry-Wert gefunden wird, fällt die Logik auf `0.0.0.0` zurück (statt Assembly-Fallback), damit Updates nicht fälschlich unterdrückt werden.

## Technische Änderung

- Kernanpassung in `apps/desktop-updater-core/UpdateOrchestrator.cs`.

---

Diese Version verbessert die Zuverlässigkeit der manuellen und automatischen Update-Erkennung deutlich.
