# Release Notes v0.0.43

Release-Datum: 25.02.2026
Tag: `v0.0.43`

## Kurzüberblick

- Ein Fehler im Bootstrapper wurde behoben: Das Installerfenster wird wieder angezeigt.
- Der Bootstrapper läuft nicht mehr im stillen Modus, wodurch das Download-/Installationsfenster sichtbar bleibt.
- Der Fix wurde lokal mit NSIS-Build und Prozess-/Fensterprüfung verifiziert.

## Installer

- `SilentInstall` im NSIS-Bootstrapper von `silent` auf `normal` umgestellt.
- Start des Offline-Setups auf `Exec` umgestellt.
- Zusätzliche Fehlermeldung ergänzt, falls das Setup nicht gestartet werden kann.
