# Release Notes v0.0.56

Release-Datum: 26.02.2026
Tag: `v0.0.56`

## Kurzüberblick

- Der Live-Installer zeigt kein separates kleines Fenster mehr.
- Nutzertexte des Bootstrapper-Ablaufs sind in das große Setup-Fenster integriert.
- Der Installationsfluss wirkt dadurch einheitlicher und ohne überlappende UI.

## Installer / UX

- NSIS-Bootstrapper läuft jetzt ohne sichtbare eigene Oberfläche.
- Das Haupt-Setup informiert im Willkommenstext über den Live-Installer-Download und die Installation.

## Technische Änderung

- Anpassungen in:
  - `installer/nsis/PCWaechterBootstrapper.nsi`
  - `installer/nsis/PCWaechterSetup.nsi`

---

Diese Version vereinfacht den Live-Installationsablauf auf ein einziges sichtbares Setup-Fenster.
