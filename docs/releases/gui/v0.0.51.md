# Release Notes v0.0.51

Release-Datum: 26.02.2026
Tag: `v0.0.51`

## Kurzüberblick

- Die Anzeige der installierten Version im Update-Check wurde korrigiert.
- Statt `v0.0.0.0` wird jetzt die tatsächlich installierte Version zuverlässig ermittelt.
- Der Update-Workflow gegen GitHub `latest` wurde robuster bei unterschiedlichen Installationspfaden.

## Update-Prozess

- Versionsauflösung erweitert auf:
  - Registry in `HKLM` und `HKCU`
  - Registry-Views `Registry64` und `Registry32`
  - Uninstall-Keys sowie App-Keys (`DisplayVersion` und `Version`)
- Zusätzlich werden Dateiversion und Assembly-Metadaten als Fallback genutzt.
- Versionsstrings mit Präfix (`v`) oder Metadaten (`+...`) werden korrekt normalisiert.

## Technische Änderung

- Kernanpassung in `apps/desktop-updater-core/UpdateOrchestrator.cs`.

---

Diese Version stellt sicher, dass Update-Hinweise die reale installierte Version anzeigen und verlässlicher auf neue Releases reagieren.
