# Release Notes v0.0.36

Release-Datum: 25.02.2026
Tag: `v0.0.36`

## Kurzüberblick

- Geplante Update-Aufgabe wurde auf `PCWächterAutoUpdater` umbenannt.
- Installer legt die Aufgabe jetzt immer an.
- Komponente „Automatische App-Updates aktivieren“ ist standardmäßig aktiv.
- Gewählte Installer-Vorgabe wird beim ersten App-Start als Default in die Einstellungen übernommen.

## Installer & Aufgabenplanung

- Bei Installation wird die Aufgabe erstellt (`ONLOGON`, `DELAY 2 Minuten`, `HIGHEST`).
- Ist die Komponente aktiv (Standard), wird die Aufgabe direkt aktiviert.
- Beim Uninstall wird `PCWächterAutoUpdater` weiterhin sauber entfernt.

## App-Logik

- `BackgroundUpdateTaskManager` nutzt nun den neuen Task-Namen `PCWächterAutoUpdater`.
- `SettingsManager` liest den Install-Default `AutomaticAppUpdatesDefault` aus der Registry,
  solange noch keine lokale `settings.json` vorhanden ist.
