# Release Notes v0.0.48

Release-Datum: 25.02.2026
Tag: `v0.0.48`

## Kurzüberblick

- Tray-Only-Start: Bei Autostart startet die App nur im Tray, das Hauptfenster bleibt verborgen.
- APP_NAME-Platzhalter im Installer wurde korrigiert, sodass überall „PC Wächter“ angezeigt wird.

## Installer

- NSIS-Setup verwendet jetzt einheitlich `APP_NAME` und `APP_TITLE` mit „PC Wächter“.
- Release- und Branding-Texte sind konsistent.

## Desktop-GUI

- Autostart erkennt den Start aus dem Windows-Startup-Ordner und zeigt nur das Tray-Icon.
- Hauptfenster wird nur bei manuellem Start angezeigt.

---

Diese Version behebt Branding-Probleme und verbessert das Nutzererlebnis beim automatischen Start.
