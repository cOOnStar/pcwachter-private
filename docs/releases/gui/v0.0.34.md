# Release Notes v0.0.34

Release-Datum: 25.02.2026
Tag: `v0.0.34`

## Kurzüberblick

- Einstellung **„Automatische App-Updates“** wurde von der Updates-Seite in die Einstellungen verschoben.
- Die Option **„Verwaiste App-Dateien bei Auto-Update bereinigen“** wurde ebenfalls in die Einstellungen verlagert.
- Fehler beim Aktivieren/Deaktivieren der automatischen Updates über `schtasks` wurde robust behoben.

## Einstellungen

- Neue Platzierung der Auto-Update-Option in der Settings-Ansicht.
- Änderungen werden dort zentral gespeichert und angewendet.
- UI-Verhalten für abhängige Optionen (Cleanup bei Auto-Update) ist konsistent gekoppelt.

## Updates-Seite

- Fokus liegt jetzt auf manueller Suche und Installation.
- Alte Auto-Update-Steuerelemente wurden aus der Updates-Seite entfernt.

## Technischer Fix

- `BackgroundUpdateTaskManager` nutzt jetzt `ProcessStartInfo.ArgumentList` statt manuell zusammengesetzter Argument-Strings.
- Dadurch werden Quoting-/Parsing-Probleme bei `schtasks.exe` vermieden.
