# Release Notes v0.0.35

Release-Datum: 25.02.2026
Tag: `v0.0.35`

## Kurzüberblick

- Einstellungen-Seite visuell überarbeitet und Struktur in klarere Bereiche aufgeteilt.
- Versionsanzeige nutzt jetzt bevorzugt die installierte Version aus der Registry.
- Manuelle Update-Suche in den Einstellungen verwendet nun die echte Update-Logik statt Simulation.
- Automatische Updates können ausstehende Downloads beim App-Start vor dem UI installieren.
- Uninstaller entfernt die geplante Auto-Update-Task und kann optional Benutzerdaten mit löschen.

## Einstellungen & UI

- Buttons im gesamten Programm kontrastreicher gestaltet.
- In den Einstellungen wird der automatische Update-Zustand angezeigt:
  - Statusmeldung
  - installierte/verfügbare Version
  - Zeitpunkt der letzten Prüfung

## Update-Verhalten

- `Nach Update suchen` in den Einstellungen ist an `AppUpdateService` angebunden.
- Bei verfügbarem Update wird der Installer wie vorgesehen heruntergeladen und gestartet.
- Bei aktiviertem Auto-Update wird ein bereits heruntergeladenes Update vor dem normalen Programmstart installiert.

## Uninstaller

- Beim Entfernen der App wird die Aufgabe `PCWaechterAutoUpdater` gelöscht.
- Neue Abfrage im Uninstall-Prozess, ob auch Benutzerdaten gelöscht werden sollen.
- Für Silent-Uninstall ist zusätzlich `/PURGEUSERDATA` möglich.
