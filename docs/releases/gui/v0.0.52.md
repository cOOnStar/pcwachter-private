# Release Notes v0.0.52

Release-Datum: 26.02.2026
Tag: `v0.0.52`

## Kurzüberblick

- Update-Installation läuft jetzt verbindlich über den geplanten Task mit höchsten Rechten.
- Direkter Installer-Fallback wurde entfernt, um wiederholte UAC-Prompts im Updatepfad zu vermeiden.
- Versions- und Installationswerte werden nach erfolgreichem Setup weiterhin in die Registry geschrieben.

## Update-Prozess

- GUI-Update stößt den Task `PCWaechterAutoUpdater` an (`RL HIGHEST`).
- Der Worker übernimmt Download/Installation still und nutzt denselben Core-Flow.
- Nach erfolgreicher Installation bleibt die Erkennung neuerer Versionen zuverlässig, da `DisplayVersion` und `InstallDir` aktualisiert werden.

## Registry / Installer

- `DisplayVersion` wird bei Installation auf die neue App-Version gesetzt.
- `InstallDir` bleibt konsistent aktualisiert.
- Autostart-/Task-Setup bleibt auf erhöhten Rechten für den automatischen Updateablauf.

---

Diese Version stabilisiert den Updatepfad hinsichtlich Rechteverwaltung, UAC-Verhalten und Versionserkennung.
