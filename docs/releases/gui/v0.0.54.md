# Release Notes v0.0.54

Release-Datum: 26.02.2026
Tag: `v0.0.54`

## Kurzüberblick

- Installer-Workflow repariert (korrekte Übergabe der Sprachressourcen-Parameter im CI-Build).
- Live-Installer bleibt auf `latest` ausgerichtet.
- Bootstrapper-/Setup-Fensterablauf und Paketgröße-Optimierungen aus der Vorversion bleiben enthalten.

## Build / CI

- `dotnet publish`-Parameter für `SatelliteResourceLanguages` werden korrekt gequotet.
- Dadurch bricht der Windows-Runner nicht mehr mit `en is not recognized` ab.

## Installer

- Offline- und Live-Installer-Build laufen wieder erfolgreich durch.
- Verkleinerte Payload-Einstellungen bleiben aktiv (`de;en`, keine Debugsymbole).

Diese Version stellt die Release-Pipeline für den Installer wieder stabil her.
