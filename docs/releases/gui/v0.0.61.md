# v0.0.61

## MVP: Agent -> API -> Console Datenfluss

- Neues FastAPI MVP (`pcwaechter-api`) mit Endpunkten:
  - `POST /agent/register`
  - `POST /agent/heartbeat`
  - `POST /agent/inventory`
  - `GET /console/devices`
  - `GET /console/devices/{id}`
  - `GET /console/devices/{id}/inventory/latest`
- Postgres-Modelle `devices` und `device_inventory` inkl. JSONB-Payload.
- Neuer Minimal-Agent (`pcwaechter-agent`) sendet Register/Heartbeat/Inventory und persistiert `device_install_id`.
- Desktop-GUI Dashboard zeigt Geräteübersicht (DataGrid + Refresh) und Latest Inventory Details.
- Docker-Setup für FastAPI MVP ergänzt (`docker-compose.yml`, `pcwaechter-api/Dockerfile`).
