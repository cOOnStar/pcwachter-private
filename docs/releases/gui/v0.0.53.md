# Release Notes v0.0.53

Release-Datum: 26.02.2026
Tag: `v0.0.53`

## Kurzüberblick

- Live-Installer bleibt auf `latest` ausgerichtet.
- Bootstrapper blendet sich vor Start des Offline-Setups aus, um Fensterüberlagerung zu vermeiden.
- Installer-Payload wurde reduziert (weniger unnötige Dateien im Zielordner).

## Installer / UX

- NSIS-Bootstrapper nutzt weiterhin `releases/latest/download/PCWaechter_offline_installer.exe`.
- Vor dem Start des eigentlichen Setups wird das Bootstrapper-Fenster ausgeblendet (`HideWindow`).

## Paketgröße / Inhalt

- Zusätzliches Brand-Icon wird nicht mehr in den Installationsordner kopiert.
- Verknüpfungs- und Display-Icons verweisen direkt auf die App-EXE.
- Publish wurde für Setup-Build gestrafft (`DebugSymbols=false`, `DebugType=None`, `SatelliteResourceLanguages=de;en`).

---

Diese Version verbessert den Installationsablauf und reduziert unnötige Daten im Installationsverzeichnis.
