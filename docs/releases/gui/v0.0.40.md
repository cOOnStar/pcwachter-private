# Release Notes v0.0.40

Release-Datum: 25.02.2026
Tag: `v0.0.40`

## Kurzüberblick

- Projektordner wurden vereinheitlicht: `desktop-gui`, `desktop-updater-worker`, `desktop-updater-core`.
- Schreibweise für den neuen Worker-Namen wurde konsistent auf `desktop-updater-worker` korrigiert.
- Die geplante Update-Aufgabe startet jetzt den separaten Updater-Worker statt der Desktop-GUI.

## Architektur & Installer

- `desktop-app` wurde in `desktop-gui` umbenannt.
- `updater-worker` wurde in `desktop-updater-worker` umbenannt.
- `update-core` wurde in `desktop-updater-core` umbenannt.
- Build-/Installer-Pfade und Solution-Referenzen wurden auf die neuen Ordnernamen umgestellt.
